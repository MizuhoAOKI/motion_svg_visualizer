# Motion SVG Visuzlizer

## How to install
Recommended version: Python 3.x
```
$ git clone https://github.com/MizuhoAOKI/motion_svg_visualizer.git
$ cd motion_svg_visualizer
$ pip install -r requirements.txt
```

## How to run
```
$ cd script
$ python gen_svg_anime.py
```
